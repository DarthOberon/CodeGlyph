# CodeGlyph

## Problem
Colleges still use outdated tools like Turbo C to enforce 
"no autocomplete" learning philosophy, but these tools are 
clunky and don't transition students to modern dev environments.

## Solution
A modern code editor with policy-driven, progressive 
unlocking of IntelliSense/autocomplete — giving teachers 
control over what assistance students get, when.

## Planned Tech Stack
- ASP.NET Core (backend)
- Monaco Editor (frontend editor)
- SignalR (real time policy updates)
- SQL Server (data)
